# Pick Website Hosting

A comprehensive website hosting comparison and review site.

## Features

- Hosting provider reviews and comparisons
- Website builder reviews
- Blog articles about hosting
- Coupon codes and deals
- Mobile-responsive design

## Structure

- `/reviews/` - Hosting provider reviews
- `/reviews/builders/` - Website builder reviews
- `/blog/` - Blog articles
- `/go/` - Affiliate redirect pages
- `/css/` - Stylesheets
- `/js/` - JavaScript files
- `/images/` - Image assets

## Deployment

This site is deployed to Hostinger via GitHub integration.
